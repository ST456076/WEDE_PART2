## Project Overview

The Clothing Marketplace System is a web-based e-commerce application developed as part of the POE project. The purpose of this project was to create an online marketplace platform where customers can browse clothing products, manage shopping activities, and interact with sellers, while administrators manage the overall system.

The project demonstrates the implementation of front-end development, backend programming, database management, user experience design, and teamwork. The system was designed to support three main users:

- Customers
- Sellers
- Administrators

Customers can browse products, add items to their cart, save items to a wishlist, and complete purchases. Sellers can upload and manage clothing listings, while administrators can manage users and monitor marketplace activities.

This project allowed the team to apply software development principles, database concepts, and problem-solving skills to create a complete marketplace solution.

# Database Design

The system uses a MySQL database to store and manage information.

Main database tables include:

## Users Table

Stores customer and account information.

## Sellers Table

Stores seller information and product ownership.

## Products Table

Stores clothing product details.

## Cart Table

Stores customer shopping cart information.

## Orders Table

Stores completed purchases.

## Requests Table

Stores marketplace-related communication and requests.

The database structure allows different parts of the system to communicate efficiently.
